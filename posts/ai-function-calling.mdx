Recently at Smol AI, we've been working on a chat interface that uses OpenAI's
Function Calling API to, at its own discretion, call functions to fetch data to
enrich its responses. It's a cool feature that opens up a lot of possibilities
for AI-powered chatbots, but working with it is not without its challenges. One
of the hazards of working on the bleeding edge of AI tech is that the
documentation is often a work-in-progress
